# quant_finance_python

A collection of quantitative finance tools built from scratch in pure Python — no NumPy, no Pandas, no external libraries.

Every formula is implemented from first principles.

---

## Tools

| File | What it does |
|------|-------------|
| `portfolio_calculator.py` | Returns, variance, covariance matrix, Sharpe ratio, efficient frontier |
| `IRR.py` | Internal Rate of Return via linear interpolation |
| `annuityfactor.py` | Annuity factor calculator |
| `dollar_cost_averaging.py` | How many shares to buy to reach a target average cost |
| `Money_market_fund.py` | Daily compounding money market simulator |
| `portfolio_rebalancer.py` | Rebalances a 60/40 stock-bond portfolio |
| `millionaire_journey.py` | Months to $1M given initial investment, rate, and contributions |
| `cumulative_return.py` | Cumulative return from a series of monthly returns |
| `daily_return.py` | Daily returns from a price series |
| `simple_moving_avg.py` | SMA with trend detection (uptrend / downtrend / mixed) |
| `max_profit.py` | Optimal buy/sell points from a price series |
| `expected_value.py` | Expected value simulator |

## Philosophy

No libraries. No shortcuts. Every calculation written by hand to understand what's happening under the hood.
